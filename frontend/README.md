# GadgetFlex — Buy Gadgets, Pay Over Time

Buy premium gadgets with flexible installment payment plans. Laptops, phones, tablets & accessories with instant approval and zero hidden fees.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/Shubbin/Gadget-pay.git

# Step 2: Navigate to the project directory
cd gadget-installments

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the project for production:

```sh
npm run build
```
